Ubuntu防火墙管理

2021-05-28 1 min read # Ubuntu

系统版本: ubuntu18.04

查看防火墙当前状态

sudo ufw status

开启防火墙

开启时请以前打开22端口的权限，否则无法ssh到机器

sudo ufw enable

ufw启用/关闭/重载

ufw enable/disable/reload

允许外部访问80端口

sudo ufw allow 80

禁止外部访问80端口

sudo ufw delete allow 80

允许此IP访问所有的本机端口

sudo ufw allow from 192.168.1.1

禁止外部访问smtp服务

sudo ufw deny smtp

安装ufw

sudo apt-get install ufw
