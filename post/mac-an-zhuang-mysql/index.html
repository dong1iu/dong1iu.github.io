mac安装MySQL

2021-05-05 1 min read # 数据库

一、下载MySQL

官网地址

下载macOS版本的dmg文件

一路下一步，同意

设置密码

二、修改环境变量

修改配置文件

sudo vim ~/.bash_profile

在配置文件中添加以下内容

# mysql
export PATH=$PATH:/usr/local/mysql/bin

alias mysql='/usr/local/mysql/bin/mysql'
alias mysqladmin='/usr/local/mysql/bin/mysqladmin'

检查文件是否生效

source ~/.bash_profile

三、测试数据库

查看MySQL版本

mysql -V

连接数据库

mysql -u root -p

退出连接的数据库

exit
