MAC下为所有人安装anaconda3

2021-05-28 1 min read # mac # python

下载安装包

  1. 在官网档案下载
  2. 在清华镜像源下载
  3. 直接下载2021年5月 Python3.8版本

安装anaconda3

安装过程中，选择为所有人安装

添加环境变量

  1. 确认anaconda3的安装位置，一般在/opt/anaconda3/bin
  2. 添加环境变量

vim ~/.zshrc

 export PATH="/opt/anaconda3/bin:$PATH"

source ~/.zshrc
